# MVP Scope

## In scope
1. Daily adaptive session generation
2. Articulation drills
3. Read-aloud fluency drills
4. Impromptu speaking practice
5. Interview simulation baseline
6. STT transcription and TTS responses
7. LLM-generated coaching feedback
8. Audio record and playback
9. Scores, levels, stages, progression
10. Weekly review dashboard (baseline)

## Out of scope for MVP
- Enterprise multi-seat dashboards
- Full custom rubric authoring UI
- Multi-language localization
